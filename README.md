# Admin Dashboard

A simple admin dashboard built as a learning project for The Odin Project. This repository demonstrates basic layout and styling techniques using plain HTML and CSS.

## Purpose
- Practice responsive layout with CSS Grid
- Build reusable UI components (cards, sidebar, header)
- Learn structuring a small frontend project

## Files
- `index.html` — main markup
- `style.css` — styling and layout
- `assets/` — icons and images
